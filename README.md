# acoustic-data-analysis-lab
Se presenta un análisis de datos acústicos experimentales obtenidos a partir de la caracterización del patrón de radiación de una fuente omnidireccional dodecaédrica, utilizando una cámara anecoica. El proyecto combina procesamiento numérico y visualización de datos, siguiendo criterios establecidos por las normas ISO 140-3 e ISO 3382-2.
