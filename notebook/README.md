# Reporte final

Esta carpeta contiene el **reporte final del proyecto de análisis acústico**, elaborado en **LaTeX** y exportado a **formato PDF**. El documento presenta de manera formal y estructurada el desarrollo completo del estudio, desde el contexto experimental hasta las conclusiones.

---

## Contenido del reporte

El reporte incluye:

- Introducción y **marco teórico** del problema acústico.
- Descripción de la **metodología experimental** empleada en cámara anecoica.
- Explicación del **procesamiento de datos** a partir de las mediciones crudas del sonómetro.
- Análisis del **patrón de radiación acústica** de la fuente omnidireccional.
- Resultados presentados mediante **espectros sonoros**, diagramas polares y gráficas de desviación.
- Discusión de resultados y **conclusiones finales**.

El formato del documento sigue el estilo de un **reporte académico**, priorizando claridad, rigor y reproducibilidad.

---

## Relación con los datos y scripts

El reporte se basa en:

- Datos crudos almacenados en la carpeta `raw/`.
- Datos limpios y consolidados en la carpeta `processed/`, obtenidos mediante el script `extraer.m`.
- Resultados y visualizaciones generadas con el script `polar_rad_multi.m`.

Esta separación permite mantener un flujo de trabajo ordenado entre datos, procesamiento y documentación.

---

## Formato y compilación

- El documento fue escrito en **LaTeX**.
- El archivo PDF incluido corresponde a la **versión final compilada** del reporte.
- En caso de recompilación, se recomienda utilizar una distribución estándar de LaTeX.

---

## Uso recomendado

- Utilizar este PDF como **referencia principal** del proyecto.
- Citar el reporte para conocer el desarrollo completo del análisis.
- Para reproducir resultados, consultar los datos y scripts contenidos en el resto del repositorio.

---

## Contexto académico

Este reporte forma parte de una **práctica experimental de acústica**, y se incluye en este repositorio como evidencia del análisis de datos aplicado a un problema científico real, integrando medición, procesamiento, visualización e interpretación.
